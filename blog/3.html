MBTI와 강아지 성격

강아지 성격을 품종별로 나눠 MBTI와 매치시킨 표입니다. 자신의 MBTI와 강아지를 매치시켜 보시고 특징을 확인하세요

1- ISTJ
강아지 종류 : 페키니즈
특징 : 침착하고 자신감 있음

2- ISFJ
강아지 종류 : 저먼 셰퍼드
특징 : 용맹하고 충성심있음

3- INFJ
강아지 종류 : 오스트레일리언 셰퍼드
특징 : 강아지들의 에너자이저

4- INTJ
강아지 종류 : 잭 러셀 테리어
특징 : 영리하고 민첩함

5- ISTP
강아지 종류 : 로디지안 리지백
특징 : 활동적이고 모험심 많음

6- ISFP
강아지 종류 : 시바견
특징 : 느긋하고 여유로움

7- INFP
강아지 종류 : 티베탄 테리어
특징 : 애정 가득함

8- INTP
강아지 종류 : 차우차우
특징 : 똑똑하고 영리함

9- ESTP
강아지 종류 : 비글
특징 : 활동적이고 호기심 많음

10- ESFP
강아지 종류 : 웰시코기
특징 : 열정적이고 에너지 넘침

11- ENFP
강아지 종류 : 레브라도 리트리버
특징 : 활동량 많고 모험심 많음

12- ENTP
강아지 종류 : 프렌치 불독
특징 : 혈기왕성하고 고집있음

13- ESTJ
강아지 종류 : 시베리안 허스키
특징 : 활동적이고 리더쉽있음

14- ESFJ
강아지 종류 : 비숑 프리제
특징 : 사교적이고 친근함

15- ENFJ
강아지 종류 : 골든 리트리버
특징 : 순하고 선함

16- ENTJ
강아지 종류 : 티베탄 마스티프
특징 : 카리스마 넘치고 똑똑함
